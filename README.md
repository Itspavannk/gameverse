# 🎮 GAMEVERSE

A premium futuristic gaming platform built with **React**, **Vite**, **Framer Motion**, and **Three.js**, featuring immersive animations, responsive layouts, and a modern gaming-inspired UI.

---

## ✨ Features

- 🎮 Premium gaming-inspired UI/UX
- 🌌 Animated Hero with Galaxy background
- 🕹 Interactive 3D Floating Controller
- 🎥 Featured Games Showcase
- 🏆 Top Games Grid
- 🚀 Upcoming Releases Section
- 💻 Platform Universe
- ❓ Animated FAQ
- 📧 Newsletter Subscription
- 📱 Fully Responsive Design
- ⚡ Smooth Scroll Navigation
- 🔝 Scroll To Top Button
- 🎨 Glassmorphism & Neon Effects
- ✨ Framer Motion Animations

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Itspavannk/gameverse.git
```

Go into the project

```bash
cd gameverse
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build production version

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
